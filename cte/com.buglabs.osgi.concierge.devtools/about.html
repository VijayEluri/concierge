Concierge Tools for Eclipse (CTE)

Concierge Tools for Eclipse (CTE) consist of a set of plug-ins that facilitate the creation and launch of OSGi applications on Concierge.

OSGi Overview

OSGi is a service oriented and component based framework specification. OSGi applications tend to be extensible and scalable. For more information please visit http://www.osgi.org.

Concierge Overview

Concierge is a highly optimized implementation of the OSGi R3 specification targeted towards mobile and embedded devices. For more information please visit http://concierge.sourceforge.net.
